Pon aqui los logos de usuario que se podran seleccionar en la tienda web.

Recomendaciones:
- Usa PNG con fondo transparente.
- Si puedes, manten un formato cuadrado.
- Tamano recomendado: 256x256 o superior.
- Usa nombres simples, por ejemplo:
  - logo-diamante.png
  - logo-fuego.png
  - logo-luna.png

Esta carpeta queda reservada para el sistema de logos de usuario.
